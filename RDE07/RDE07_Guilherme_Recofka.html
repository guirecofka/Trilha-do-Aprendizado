Branches

São ramificações de um projeto de software ou documento e tem como objetivo o desenvolvimento paralelo de:

  • Novas funcionalidades;
  • Correção de problemas;
  • Alteração de código.

Hospedagem

o GitHub Pages oferece hospedagem gratuita de sites. Basta criar uma conta e um repositório público e a URL será:https://nomeusuario.github.io/repositorio

Para ser hospedado no GitHub é necessário clicar no repositório que você deseja hospedar, clica em configurações e rola a página até "GitHub Pages". A licença do repositório deve ser pública e clicando em None você escolhe a branch principal; salvando as configurações. Dessa forma, automaticamente seu site é públicado e mostra o seu site.
