INS

A tag INS é, em suma, uma indicação de trecho inserido. A implementação no código é simples e basta adicionar entre as tags o trecho desejado.

DEL

A tag DEL é, em suma, uma indicação de trecho removido/deletado. A implementação no código é simples e basta adicionar entre as tags o trecho desejado.

SUB

BDO

A tag BDO é usadA para substituir a direcionalidade atual do texto. Isso faz com que a direcionalidade dos personagens seja ignorada em favor da direcionalidade especificada. Possui os atributos globais ltr e rtl. Exemplo:

CODE

A tag CODE apresenta seu conteúdo estilizado de maneira a indicar que o texto é um pequeno fragmento de código. Por padrão, o conteúdo é exibido utilizando a fonte monoespaçada padrão do user agent's (en-US). Exemplo: A função selectAll() destaca todo o texto no campo de entrada então o usuário pode, por exemplo, copiar ou deletar o texto.

PRE

A tag PRE é, em suma, uma indicação de definição de um texto pré-formatado.

KBD

A tag KBD é, em suma, uma indicação de definição de um texto do próprio teclado
